# clojure-benchmark

A simple library for benchmarking bits of clojure code.


## Utilities

Generate HTML documentation:

```
$ lein codox
```

Run tests (once):

```
$ lein spec
```

Run Tests (auto-run on-save):

```
$ lein spec -a
```

Deploy to Clojars (requires username and deploy token):

```
$ lein deploy clojars
```
